# Keeper

Esse aplicativo foi desenvolvido para ser um "To do list", onde o usuário poderá colocar suas tarefas e acompanhá-las pela plataforma.

## 💡 Objetivo

A prática na utilização de AJAX, consumo de API e local storage.
